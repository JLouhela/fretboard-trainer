# Fretboard Trainer

https://github.com/JLouhela/fretboard-trainer

A progressive web app for learning every note on the guitar fretboard.

## Features

- **Multiple exercise types** — identify notes by position, find positions by note name, or read notes from staff notation
- **All 6 strings** — individual string drills plus low strings (E, A, D) and high strings (G, B, E) groups
- **Piano-style note buttons** — sharps/flats offset above naturals, hidden automatically for natural-note exercises
- **Audio feedback** — major chord on correct answers, minor chord on wrong
- **PWA** — installable, works offline

## Tech Stack

Vite · Preact · TypeScript · vite-plugin-pwa

## Development

```bash
npm install
npm run dev
```
